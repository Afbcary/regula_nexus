Using [Python UV](https://docs.astral.sh/uv/guides/install-python/)

```bash
uv run main.py
```

This script scrapes https://usaultimate.org/rules/ and generates /regula_porro/src/rules.json.

Within the React Next project regula_porro, run locally:

```bash
npm run dev
```

Open http://localhost:3000.
